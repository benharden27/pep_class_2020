    Getting Started in R

    Our goal in this class is to introduce you to some skills in exploring oceanographic data in R. This will mainly involve plotting tools, but will also include some discussion of working with data sets.

    Why R?

    Perhaps the best place to start is to ask ourselves why we are choosing to use R. Fortunately, Rob Schlegel (WHOI) sums up nicely why using R is a good idea for data science as part of his R workshop. Please read this.

    Dan Kelley (Dalhousie) has a nice book on oceanographic analysis in R - his sub-chapter on “Why R, and Why Now?” is a quick read to help contextualize the desire to use R for oceanographic work.

    Please read both these short justifications to give you an insight into why R is something worth your energy.

    Downloading R and Rstudio

    You should next download both R (the programming language) and Rstudio (a nice visual interface that makes writing code more intuitive) and opened RStudio to see that it has worked properly.

    You need to download R first:

    • Go to http://www.r-project.org/ and choose “download R” under “Getting Started”.
    • Select any of the sites to download R from
    • Choose the R link for your operating system
    • Windows: select “base” and then “Download R for Windows”. Run the .exe file that you download to install R
    • Mac: select the .pkg link to download. Run the file that you download to install.

    Then you need to download RStudio:

    • Go to http://www.rstudio.com
    • Click “Download RStudio” and following the instructions for your operating system.

    To use R you need to open RStudio (not the R link that might have appeared on your desktop).
